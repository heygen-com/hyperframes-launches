01 — The Dial0:00 – 6:00
White screen. Single horizontal line of black text: "You use HyperFrames to make the perfect _____ video." The blank word is a split-flap dial that cycles through video types — launch, explainer, training, product demo — and lands on "UGC ad". All text is the same black Inter 800. No pill, no badge, no color on the dial word — it's inline with the sentence. The rotating words are the same typeface as the rest of the line.
Dial mechanic: Incoming word rotates in from below on the Z-axis, perspective-bent toward the viewer (rotateX ~65deg on the leading edge). The outgoing word flips up and backward to the same bend. Words at 0.7 opacity when they're in the bent/waiting position, full opacity when flat in the line. Each flip takes ~0.5s. Cycle through 4-5 words before landing.
Landing: When "UGC ad" snaps into place, the last outgoing word ("explainer" or similar) remains visible above at ~0.35 opacity, tilted back, then fades to 0. Brief hold (~0.8s) with the full line reading cleanly before transitioning to Scene 2.
Typography: Inter 800 for everything. All black (#0a0a0a). Single horizontal line, vertically centered on white.

04 — The Variables16:00 – 23:00
The original ad is center-frame in a full-size vertical glass card. Two more glass cards slide out from behind it — one left, one right — all three filling the screen at equal size. The left and right cards have the variable code overlaid on them (dark semi-transparent code block on the glass surface). Changed values are highlighted (red on left, blue on right) and flip using the same split-flap dial mechanic from Scene 1. After the values flip, the code fades away and the variant videos are revealed underneath on the same cards.
Phase 1 (1.5s): Two glass cards slide out from behind the center card, filling the screen as three equal-size vertical cards. White background throughout.
Phase 2 (2.5s): Code overlays appear on the left and right cards — dark semi-transparent panels with the variable key/value pairs. Changed values highlight and flip using the rotateX split-flap — same Z-axis bend as Scene 1, same 0.7 opacity. hookText and accentColor are the most readable flips.
Phase 3 (1s): Code overlays fade away, revealing the variant videos underneath on the same glass cards. Three full-size vertical glass cards with different videos now on screen.
Code overlay: ~6 lines each. JetBrains Mono on a semi-transparent dark panel (rgba black ~0.75). Keys in muted gray, values in light, changed values get colored highlight pills.
